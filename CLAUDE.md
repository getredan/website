# redan-website

Static website for redan. Single HTML file, no build step, no framework.

Follows Simon Willison's HTML tools philosophy: one file, inline CSS/JS,
load fonts from CDN, keep it small.

## Structure

- `index.html` -- the entire site
- `logo.jpg` -- redan watercolor logo

## Deploy

Static hosting. Push to GitHub, serve via GitHub Pages or similar.

## Style

- Font: DM Serif Display (headings), Inter (body), JetBrains Mono (code)
- Palette: warm stone tones from the watercolor logo
- No React, no build step, no npm
